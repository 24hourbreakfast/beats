# Bern Ban Beats

This repo powers `bernban.com/beats` and includes:

- Vaporwave-styled beat page with a cleaner player
- Newest tracks first (from `beats/tracks.json`)
- Automatic local watch-folder uploads to GitHub

## Site

- Page: `beats/index.html`
- Audio files: `beats/*.mp3`
- Manifest: `beats/tracks.json`

`tracks.json` is generated by:

```bash
python3 tools/generate_tracks_manifest.py
```

## Auto Upload Setup (macOS)

1. Install the background uploader:

```bash
./tools/install_launch_agent.sh "/Users/nathanbernier/Music/BeatsDrop"
```

2. Drop new `.mp3` files into that watch folder.
3. The uploader will:
- Copy file into `beats/` with a timestamp prefix
- Regenerate `beats/tracks.json`
- Commit + push to `main`
- Move source file into `<watch-folder>/_uploaded`

### Logs

- `.uploader_state/launchd.out.log`
- `.uploader_state/launchd.err.log`

### Stop/Start Service

```bash
launchctl unload ~/Library/LaunchAgents/com.bernban.beats-uploader.plist
launchctl load ~/Library/LaunchAgents/com.bernban.beats-uploader.plist
```

## Netlify Setup (new account)

Create a new Netlify account (GitHub login is fine), then:

1. `Add new site` -> `Import an existing project` -> select `24hourbreakfast/beats`
2. Branch: `main`
3. Build command: leave empty
4. Publish directory: `.`
5. Deploy

`netlify.toml` is included and sets deploy defaults.

After deploy, add custom domain `bernban.com` in Netlify.
You must have access to your DNS registrar to point the domain to Netlify.

## AB Visual Batch Generation

Generate a full AB art pack in one command:

```bash
export OPENAI_API_KEY="your_key_here"
./tools/generate_ab_visuals.sh
```

Prompt config:

- `ab/visual-prompts.json`

Output:

- PNGs in `assets/ab/`
- Manifest in `assets/ab/manifest.json`
